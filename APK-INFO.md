## Como Instalar

1. Transfira o arquivo `Controle-Financeiro-v1.0.apk` para seu dispositivo Android
2. Habilite "Origens desconhecidas" nas configurações do Android
3. Toque no arquivo APK para instalar
4. Confirme a instalação

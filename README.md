## 🎯 **Visão Geral**

**Servo Tech - Finanças** é um aplicativo web de última geração para controle financeiro pessoal e empresarial. Desenvolvido com **JavaScript ES6+**, **Firebase**, **PWA** e **IA integrada**, oferece uma experiência completa de gestão financeira com recursos avançados como reconhecimento de voz, orçamentos compartilhados, análises inteligentes e muito mais.

### 🌟 **Destaques Principais**
- 🎤 **Comandos de Voz Inteligentes** - Adicione transações falando naturalmente
- 🤝 **Orçamentos Compartilhados** - Gestão financeira colaborativa
- 📊 **Analytics Avançados** - Insights detalhados sobre seus gastos
- 🔔 **Notificações Inteligentes** - Alertas personalizados em tempo real
- 📱 **PWA Nativo** - Experiência de app móvel completa
- 🌙 **Temas Adaptativos** - Interface que se adapta às suas preferências

**🌐 Aplicativo Online:** https://financas-app-819a8.web.app

## 📊 **Estrutura de Dados**

### **Transação**
```javascript
{
  id: "transaction_id",
  descricao: "Supermercado",
  valor: 150.00,
  tipo: "despesa", // ou "receita"
  categoriaId: "alimentacao",
  budgetId: "budget_id",
  userId: "user_id",
  createdAt: Timestamp,
  recorrenteId: "recorrente_id" // opcional
}
```

### **Categoria**
```javascript
{
  id: "category_id",
  nome: "Alimentação",
  tipo: "despesa", // ou "receita"
  limite: 500.00, // opcional
  cor: "#FF6B6B",
  budgetId: "budget_id",
  userId: "user_id"
}
```

### **Recorrente**
```javascript
{
  id: "recorrente_id",
  descricao: "Netflix",
  valor: 39.90,
  categoriaId: "entretenimento",
  dataInicio: "2024-01-01",
  parcelasTotal: 12,        // Total de parcelas
  parcelasRestantes: 8,     // Parcelas restantes
  ativa: true,
  efetivarMesAtual: true,   // Aplicar no mês atual
  budgetId: "budget_id",
  userId: "user_id"
}
```

## 🚀 **Instalação e Configuração**

### **Pré-requisitos**
- Node.js 14+ 
- npm ou yarn
- Conta Firebase
- Git

### **Instalação Local**

```bash
# 1. Clone o repositório
git clone https://github.com/Igor-Bispo/Financeiro-app.git
cd Financeiro-app

# 2. Instale as dependências
npm install

# 3. Configure o Firebase
# Crie um projeto no Firebase Console
# Ative Authentication (Google), Firestore e Hosting
# Copie as configurações para src/js/firebase.js

# 4. Configure as variáveis de ambiente
cp .env.development.example .env.development
# Edite .env.development com suas configurações

# 5. Inicie o servidor de desenvolvimento
npm run dev
```

### **Configuração do Firebase**

1. **Authentication**: Ative o provedor Google
2. **Firestore**: Configure as regras de segurança (use firestore.rules)
3. **Hosting**: Configure o domínio personalizado
4. **Indexes**: Importe firestore.indexes.json

## 📞 **Suporte e Troubleshooting**

### **Problemas Comuns**

**Sistema de Voz:**
- **"Nenhum microfone encontrado"** → Use a entrada de texto
- **"Permissão negada"** → Clique no ícone do microfone na barra
- **"Microfone em uso"** → Feche outros apps

**Recorrentes:**
- **Não aplica automaticamente** → Use o botão "Aplicar Recorrentes"
- **Parcela incorreta** → Verifique se `parcelasTotal` está salvo
- **Duplicação** → Sistema evita duplicação automática

**Categorias:**
- **Progress bar não aparece** → Verifique se há limite definido
- **Cálculos incorretos** → Recarregue a página

### **Para Dúvidas**
- Verifique os logs do console (F12)
- Teste em diferentes navegadores
- Consulte a documentação técnica
